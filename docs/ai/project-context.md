# Project Context

Project: Expense & Budget Manager

Stack:
- Laravel (backend)
- Blade templates
- Livewire for interactivity
- Filament for admin dashboard

Goal:
Learn Laravel ecosystem with a real-world SaaS-style project.

Users can:
- manage categories
- track transactions
- set budgets
- see analytics dashboards

Architecture focus:
- clean Laravel structure
- Eloquent relationships
- Livewire components
- Filament resources
